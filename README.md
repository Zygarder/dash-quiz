# Dash Quiz

A web-based system designed to assess and monitor the readiness of senior high school students under the CSS strand for NC II certification. The platform provides quizzes, performance tracking, and analytics to support both students and teachers in identifying strengths and areas for improvement.

## Features
- User authentication (Admin & Student)
- Quiz management system
- Performance analytics dashboard
- NC II readiness tracking
